    Roadmap to become Web Developer


                  A web developer is a programmer who specializes in the creation, design, and maintenance of websites and web applications. In this simple blog post, i have written step by step process with resoureces like frameworks and tools.


    Talking about Website, there are 3 components:-


    1. Front-End


    2. Back-End


    3. Database



    Step 1. Front-End~

           It is the client side interface of website, Where we need to learn 3 languages;


    1. HTML:-

           It is a markup language which is used to build structure of the web page. It is easier so you can learn in short time.


    2. CSS:-

           It is style sheets which used to design HTML content and make page attractive. You don't need to master at first but need to learn basics like color, border, background, margin, padding, id, class, etc.


    3. JavaScript:-

           It is a programming language which is used to add functionalities into page. It can be little harder for beginners.



    Step 2. Bootstrap and Projects~


    1. Bootstrap:-

           It is a free and open-source CSS framework where you can get HTML, CSS and JavaScript-based design templates for forms, buttons, navigation, and other interface components. It makes coding easier and also save your valuable time.

    Bootstrap link:~ Bootstrap


    2. Projects:-

        You can make projects from Front-End only like, Clone website of popular sites such as facebook, instagram, twitter and many. Also you can make portfolio website of your own.


    You can lookout mine Portfolio website (HTML, CSS, JavaScript and Bootstrap):~ Portfolio Website



    Step 3. Back-End~

           Time to explore Back-End, It is the server side of any web applications. There are many languages for Back-End but you need to learn one;


    -NodeJs (JavaScript)

            My personal recommendation is NodeJs which is most popular option and based on JavaScript. So, we can use JavaScript as both Front-End and Back-End.

    -Django (Python)

    -PHP

    -Ruby

    -Java



    Step 4. Database~

            It is an organized collection of data for the application. There are also many options;


    -MongoDB

            My personal recommendation is MongoDB which is written in JavaScript.

    -MySQL

            written in C, C++.

    -SQlite

            Written in C but you can use for Python also.

    -Cassandra

            Written in Java

    -DocumentDB



    Step 5. libraries/Framework of Front-End~


    -ReactJs

            My personal recommendation is ReactJs because it is based on JavaScript.

    -Angular

    -Vujs



    Step 6. Others IMP's to know~


    -Git/Github

           It is very important to know, and My Github account is~ Github

    -API/Network HTTP

    -Linux

    -Domain name and Hosting

           After making website, you need to buy domain name and host your website. You also can host your website for free in many platforms. My personal recommendation is Github.


           Remember, becoming a web developer is an ever-evolving process. Must be stay up to date with the latest trends and advancements in the field. You can follow above simply mentioned details process blog post.
